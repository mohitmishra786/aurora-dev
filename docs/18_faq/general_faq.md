# Frequently Asked Questions

Common questions about AURORA-DEV.

**Last Updated:** February 8, 2026  
**Audience:** All Users

## General

### What is AURORA-DEV?
An autonomous multi-agent system that builds production-ready software from natural language specifications.

### What languages does it support?
Python, TypeScript, JavaScript. More coming.

### Can I use my own API keys?
Yes, you need your own Anthropic API key.

## Usage

### How long does a project take?
Simple APIs: 15-30 min. Full-stack apps: 45-90 min.

### Can I modify generated code?
Yes, all code is yours. Iterate using `aurora modify`.

### Does it write tests?
Yes, unit, integration, and E2E tests are generated automatically.

## Cost

### How much does it cost?
Depends on project complexity. Simple API ~$5, full-stack ~$15-25.

### Can I set budget limits?
Yes, via `cost_management.daily_budget` in config.
