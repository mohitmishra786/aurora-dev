## Developer Salary

Average Senior Dev: $150k/year (~$75/hr).
If Aurora saves 10 hours/week (25%), that is $37,500/year in value.

## Cost of Aurora

Enterprise Plan: $500/month/user ($6,000/year).
API Costs: ~$100/month.
Total: $7,200/year.

## Net Gain

Value ($37.5k) - Cost ($7.2k) = **$30,300 per developer per year.**

## Intangibles

- Faster Time-to-Market = Revenue acceleration.
- Higher Code Quality = Reduced bug fixing costs.
