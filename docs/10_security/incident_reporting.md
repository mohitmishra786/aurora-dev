# Incident Reporting

See something, say something.

**Last Updated:** February 8, 2026
**Audience:** All Employees

> **Before Reading This**
>
> You should understand:
> - [Incident Response](../09_operations/incident_response.md)

## How to Report

If you suspect a breach:
1. **Slack:** `#security-incidents` (Private channel).
2. **Email:** `security@aurora.dev`.

## What to Report

- Phishing emails.
- Accidental commit of API keys.
- Strange server behavior.
- Laptop loss/theft.

## Whistleblower Protection

Reports can be made anonymously via our third-party hotline.
We have a strict non-retaliation policy.

## Bug Bounty

We operate a public Bug Bounty program on HackerOne.
Security researchers are friends, not enemies.
