# Example: Full Stack Application

Building a complete full-stack app with AURORA-DEV.

**Last Updated:** February 8, 2026  
**Audience:** All Users

## Prompt

```markdown
Build a social reading application.

Features:
- User profiles with reading history
- Book search and discovery
- Reading lists/shelves
- Reviews and ratings
- Social follows and activity feed
- Reading challenges

Tech: FastAPI, React, PostgreSQL
```

## Generated Components

### Backend
- FastAPI with async handlers
- SQLAlchemy models
- Pydantic schemas
- JWT authentication

### Frontend
- React 18 with TypeScript
- Zustand state management
- Tailwind CSS styling
- SWR data fetching

### Database
- PostgreSQL with pgvector
- Full-text search
- Recommendation engine

## Quality
- 85% test coverage
- Responsive design
- Accessible (WCAG 2.1)
