## Pre-Flight Check

### Security
- [ ] MFA enabled for all admins.
- [ ] TLS 1.3 verified.
- [ ] Secrets rotated in last 90 days.
- [ ] Penetration test passed.

### Privacy
- [ ] Cookie banner active.
- [ ] Privacy Policy linked in footer.
- [ ] Data Processing Addendum available.

### Reliability
- [ ] Backups verified (restore test passed).
- [ ] DR plan reviewed.
- [ ] Status page operational.

### Legal
- [ ] Terms of Service updated.
- [ ] Third-party licenses reviewed (no GPL violations).
