## Data Processor vs Controller

AURORA-DEV acts as a **Data Processor** for our customers.
Our customers are the **Data Controllers**.

## PII Definition

Personally Identifiable Information (PII) includes:
- Name, Email, IP Address.
- Agent conversation history (if it contains names).

## DPA

We sign a Data Processing Addendum (DPA) with all enterprise customers.
Typical Standard Contractual Clauses (SCCs) apply for data transfer to US.

## Consent

We require explicit opt-in for marketing emails.
Application usage data is collected under "Legitimate Interest" for security and billing.
