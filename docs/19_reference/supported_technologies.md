# Supported Technologies

What we play nice with.

**Last Updated:** February 8, 2026
**Audience:** Architects

## Languages

- **Python:** First-class citizen. (3.10+)
- **TypeScript/JavaScript:** First-class citizen. (Node 18+)
- **Go:** Supported via generic agent.
- **Java:** Supported via generic agent.
- **Rust:** Supported via generic agent.

## Databases

- **Postgres:** Primary supported DB.
- **MySQL:** Compatible (via SQLAlchemy).
- **SQLite:** Supported for local dev.

## Clouds

- **AWS:** Native integration (S3, RDS, EKS).
- **GCP:** Good support (GKE, Cloud SQL).
- **Azure:** Good support (AKS, Azure SQL).
