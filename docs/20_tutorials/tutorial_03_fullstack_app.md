# Tutorial 03: Fullstack App

Adding a React frontend.

**Last Updated:** February 8, 2026
**Difficulty:** Intermediate
**Time:** 20 mins

## Goal

Create a React UI for the To-Do API from Tutorial 02.

## Step 1: Task Definition

```markdown
# Objective
Create a React app using Vite in a folder named `frontend`.
It should fetch tasks from `http://localhost:8000/tasks` and display them in a list.
Add a form to create new tasks.
Style with TailwindCSS.
```

## Step 2: Run Agent

```bash
aurora run --task task.md --agent frontend
```

## Step 3: Launch

```bash
cd frontend
npm install
npm run dev
```
