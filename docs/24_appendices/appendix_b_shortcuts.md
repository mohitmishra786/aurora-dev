# Appendix B: Keyboard Shortcuts

Speed up your workflow in the TUI (Text User Interface).

**Last Updated:** February 8, 2026
**Audience:** Power Users

> **Before Reading This**
>
> You should understand:
> - [Using CLI](../05_user_guides/using_cli.md)

## Navigation (Global)

| Key | Action | Scope |
|-----|--------|-------|
| `?` | Show help menu | Global |
| `q` | Quit / Back | Global |
| `Tab` | Next pane | Dashboard |
| `Shift+Tab` | Previous pane | Dashboard |

## Task View

| Key | Action |
|-----|--------|
| `Enter` | View detailed task info |
| `r` | Retry selected task |
| `d` | Mark task done |
| `x` | Cancel task |
| `/` | Filter tasks |

## Logs View

| Key | Action |
|-----|--------|
| `f` | Follow (tail) logs |
| `c` | Clear log buffer |
| `PageUp` | Scroll up |
| `PageDown` | Scroll down |
| `Home` | Jump to top |
| `End` | Jump to bottom |

## Interactive Mode

| Key | Action |
|-----|--------|
| `Ctrl+C` | Interrupt current generation |
| `Ctrl+D` | EOF / Submit input |
| `Up` | History previous |
| `Down` | History next |
